# ToDoList
Lista de Tarefas (Afazeres)

## Ex. 01:
Agora chegou a hora de aprimorar o funcionamento da nossa aplicação!

Botão Adicionar:

Deve inserir um novo item na lista, contendo o texto digitado pelo usuário e com o checkbox desmarcado.

Deve apagar o texto digitado pelo usuário do campo de texto.

O que acontece se o usuário clicar no botão e não existir nenhum texto digitado? Escrever um código para tratar esse cenário.

## Ex. 02:
Programar a interação com o checkbox de um item da lista, para marcar/desmarcar uma tarefa como concluída:

Caso o checkbox esteja desmarcado, deve marcar o checkbox e tachar o texto do item (risco sobre o texto, “line-through”)

Caso o checkbox esteja marcado, deve desmarcar e remover o estilo de texto tachado.

## Ex. 03:
Programar o que acontece quando o usuário clicar no botão de excluir um item da lista.

Deve remover apenas o item correspondente da lista, mantendo todos os outros.

E se o usuário clicar sem querer no botão excluir? Essa remoção é definitiva, então seria uma boa ideia confirmar com o usuário se ele realmente deseja fazer isso.

## Ex. 04:
Para cada ação de adição ou exclusão de tarefas:

Deve ser calculado o número de tarefas na lista

E exibida em algum lugar da tela
